# **App Name**: Weather Parader

## Core Features:

- Current Weather Forecast: Display the current weather conditions for the user's location using browser geolocation.
- Custom Location Forecast: Allow users to manually enter a location and retrieve weather data. After selecting location and date, an optional query can be provided; an AI tool analyzes query and answers it regarding the user specified custom weather.
- Journey Weather Planner: Plan trips by inputting start and end locations and travel dates to view weather forecasts along the route. Uses a step-by-step route forecast indicating the weather along the various legs of the trip.
- Probabilistic Rain Forecast: Generate a probabilistic forecast for rain, displaying the chances of rain at different times of the day.
- 7-Day Forecast: Display a 7-day weather forecast, including weather icons, day names, conditions, temperature ranges, and precipitation percentages.
- Hourly Forecast Chart: Show an hourly forecast chart displaying temperature and precipitation probability over the next few hours.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to create a professional, dark-themed look.
- Background color: A very dark, desaturated blue (#121536), creating a gradient background effect.
- Accent color: Light blue (#748ffc) for interactive elements like icons and buttons.
- Headings font: 'Playfair', serif, for elegant and bold headlines.
- Body text font: 'Inter', sans-serif, for clean and readable descriptions.
- Use blue accent icons to represent weather conditions, location, and other features.
- Dark translucent cards with rounded corners and thin borders for a modern UI.
- Smooth hover animations for interactive elements like buttons and cards.
